# Sound Files for Pomodify

This directory contains notification sound files for the Pomodify app.

## Required Sound Files

You need to add the following MP3 files to this directory:

1. **bell.mp3** - Classic bell sound
2. **chime.mp3** - Soft chime sound
3. **digital.mp3** - Digital beep sound
4. **soft.mp3** - Soft ding sound
5. **tick.mp3** - Ticking sound for timer (optional)

## Where to Get Sound Files

### Free Sound Resources:
- **Freesound.org** - https://freesound.org/
- **Zapsplat** - https://www.zapsplat.com/
- **Mixkit** - https://mixkit.co/free-sound-effects/
- **Pixabay** - https://pixabay.com/sound-effects/

### Recommended Search Terms:
- "notification bell"
- "chime"
- "digital beep"
- "soft ding"
- "clock tick"

## File Requirements:
- Format: MP3
- Duration: 1-3 seconds (short and crisp)
- File size: < 50KB each
- Sample rate: 44.1kHz or 48kHz

## Testing:

After adding sound files, test them in the app:
1. Go to Settings > Sound Settings
2. Enable "Notification Sound"
3. Select a sound type
4. Click the "Test" button
