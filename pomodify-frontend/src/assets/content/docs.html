Documentation

This quick documentation covers common user workflows and where to find additional resources.

Getting started

Create an account, add one or more activities, and start a timed session. Use session notes to capture what you worked on.

Timers

Customize durations for focus, short breaks, and long breaks. You can pause, resume, or skip sessions as needed.

Reports & export

View recent session history and simple weekly summaries. Export session data as CSV from the Settings page.

Developer & API

For developer-facing API details and source repository information, check the project repository or contact us for access to extended docs.
