# Product Overview

Pomodify is a customizable Pomodoro productivity tracker that helps users focus with flexible timers, activity grouping, and progress tracking.

## Core Features
- Customizable work/break sessions with flexible timer settings
- User accounts with JWT authentication and OAuth2 (Google) integration
- Activity categorization system for organizing sessions
- Session logging with notes and comprehensive progress reports
- Push notifications via Firebase Cloud Messaging
- Responsive web and mobile interface
- Real-time session synchronization across devices

## Target Users
Students, professionals, and anyone looking to improve productivity using the Pomodoro Technique with enhanced tracking and customization capabilities.
